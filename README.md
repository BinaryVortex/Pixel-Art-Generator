# Pixel Art Generator

A simple, lightweight Pixel Art Generator built using HTML, CSS, and JavaScript.

![Pixel Art Generator Screenshot](./Screenshot%202024-04-26%20115202.png)

## Description

This repository contains a small web app that lets you design pixel art on a grid. Use the color picker, change the grid size, and draw pixel-by-pixel to create simple sprites and icons. The app is client-side only and requires no server — open `index.html` in a browser to run it.

## Features

- Draw pixel art on a customizable grid
- Color picker for selecting drawing color
- Toggle grid lines and clear the canvas
- Easy to run locally — no build tools required

## Demo / Preview

Open the `index.html` file in your browser to see the app in action. The included screenshot above shows the editor interface.

## Usage

1. Clone the repository:

   ```bash
   git clone https://github.com/BinaryVortex/Pixel-Art-Generator.git
   cd Pixel-Art-Generator
   ```

2. Open `index.html` in your web browser (double-click the file or use your browser's "Open File" feature).

3. Use the controls to change grid size, pick colors, draw, or clear the canvas.
